# Health Survey Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a mental health survey dataset to understand patterns related to age, gender, treatment, and work interference.

## Objectives
- Clean and preprocess survey data
- Analyze demographic distributions
- Identify treatment and work-interference trends
- Visualize key insights using plots

## Tools & Technologies
- R
- RStudio
- tidyverse (dplyr, ggplot2)
- CSV data handling

## Files Included
- EDA R script (.R / .Rmd)
- Dataset (.csv)
- Visualization outputs

## Outcome
Generated insights and visualizations
